[SIZE=6][B][ITM] Chevereto Integration for XenForo 2.0.0 Alpha+[/B][/SIZE]

[SIZE=5][B]Description[/B][/SIZE]
The popup upload plugin (PUP) is a small file that allows Chevereto based websites to provide external image uploading via a small JavaScript file.

[SIZE=5][B]Requirements[/B][/SIZE]
[LIST]
[*]PHP 5.6.0+
[/LIST]

[SIZE=5][B]Options[/B][/SIZE]

[TH]Group[/TH]

[TD][ITM] Chevereto Integration[/TD]
[TD]Auto insert[/TD]
[TD]Enable/Disable the auto insert function to let the user choose what he want to do.[/TD]
[/TR]
[TR]
[TD][ITM] Chevereto Integration[/TD]
[TD]Color palette[/TD]
[TD]The color palette for your upload button.

[/TR]
[TR]
[TD][ITM] Chevereto Integration[/TD]
[TD]Embed codes[/TD]
[TD]Embed codes to auto insert in the target editable content. Default [I]bbcode-embed-medium[/I]. Codes using [I]full[/I], [I]medium[/I] or [I]thumbnail[/I] will link to the image viewer page.[/TD]
[/TR]
[TR]
[TD][ITM] Chevereto Integration[/TD]
[TD]Button position[/TD]
[TD]Position, relativ to the sibling, to place the button[/TD]
[/TR]
[TR]
[TD][ITM] Chevereto Integration[/TD]
[TD]Upload to[/TD]
[TD]Url to your Chevereto installation. [B]Without[/B] trailing slash
E.g.: [B]www.example.com[/B] or [B]www.example.com/chevereto[/B][/TD]
